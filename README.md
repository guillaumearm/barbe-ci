SDP Pointing System
=============

### Download
```bash
$ git clone https://trapcodien@bitbucket.org/trapcodien/sdp.git
$ cd sdp
```

### Install
```bash
$ npm install
```

### Start dev server
```bash
$ npm run dev
```
Then, take your favorite browser and go to http://localhost:8080


### Start tests
##### unit tests :
```bash
$ npm run test
```
##### lint tests :
```bash
$ npm run lint
```

##### all tests :
```bash
$ npm run test:all
```

### More commands
```bash
$ npm run help
```
